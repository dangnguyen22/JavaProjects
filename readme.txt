Project done by Michael Nguyen - Texas Christian University at Fall 2022.
This project implements a GUI to allows user to create two matrices with 
any rows/cols and do mathematical addition, subtraction, multiplication 
with created matrices. Also supports saving and opening of the user's process. 

How to run: 
- Press the CREATE button to create the matrices with the desired rows and collumns
- Select the operation you desired on the combobox. 
- Press EXECUTE to execute the operation and shows the result matrix 
- If you want to save the matrix as a file, you must READ it beforehand
- Then, press SAVE and enter file name. 
- To open, press OPEN and enter file name.
